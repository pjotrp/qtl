 L.S. O'Donoughue, S.F. Kianian, P.J. Rayapati, G.A. Penner, M.E. Sorrells,
 S.D. Tanksley, R.L. Phillips, H.W. Rines, M. Lee, G. Fedak, S.J. Molnar,

 molecular linkage map of cultivated oat.  Genome 38:368-380
 
 
 The mapping population consisted of 71 F6:7 recombinant inbred lines.  The
 map was constructed using MAPMAKER V2.0 using a LOD threshold of 2 to order
 the framework markers.  Markers which could not be positioned at LOD 2 were
 assigned to the most likely interval.  True recombination estimates for RI
 data were then established on the framework map using RI Plant Manager
 V2.4, and converted to Kosambi map distances.
